HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element contains data not visible to the user
  • The list elements are ordered or unordered
  • HTML is case-sensitive, but declarations are not
  • Header elements are used to define header sections, but H1 should only be used once
  • Alt text descriptions are useful for those who need screen-reading but also optimise SEO

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Values can be determined with semantics and codes, such as colour
  • you can create style sheets and link them externally in the index

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git comes before every command
  • git ls brings up the directories available
  • git cd opens a particular directory
  • 'code .' launches a file in VS

JavaScript

    • A variable is a named container that allows us to store data in our code.
    • Control flow is the order in which a computer executes code in a script.
    • variables are stores of data
    • some variable types are strings, booleans, bigints
    • an array can store different variable types
    • you can access the values in an array through the index [0], [1] etc...
    • if and else are conditionals that run if a statement is truthy
    • for statements are run for as long as a condition is true
    • functions only run when called in the code and can be run multiple times
    • functions must be defined and declared
